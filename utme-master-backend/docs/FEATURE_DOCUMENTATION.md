#### NECO Exams
- NECO past questions

- Timed sessions

### Question Types

#### MCQ (Multiple Choice)
- Four options (A, B, C, D)
- Single correct answer
- Detailed explanations
- Difficulty levels

#### True/False
- Two options
- Simple format
- Quick assessment

#### Fill in the Blank
- Text input
- Case-insensitive
- Multiple correct answers

#### Essay
- Long-form answers
- Manual grading
- Detailed feedback

#### Matching
- Pair matching
- Multiple pairs
- Auto-scoring

### Question Configuration

#### Basic Information
- **Question Text** - Main question
- **Question Type** - Select type
- **Subject** - Select subject

- **Difficulty** - EASY, MEDIUM, HARD

- **Exam Type** - JAMB, WAEC, etc.

#### Answer Configuration
- **Options** - Add answer choices

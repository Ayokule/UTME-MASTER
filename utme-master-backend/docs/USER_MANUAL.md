## Getting Started

### What is UTME Master?

UTME Master is a comprehensive exam preparation platform designed for Nigerian students preparing for:
- **JAMB** (Joint Admissions and Matriculation Board)
- **WAEC** (West African Examinations Council)
- **NECO** (National Examinations Council)

### Key Features

- **Official Exams** - Practice with real JAMB, WAEC, NECO past questions
- **Practice Tests** - Customizable practice exams by subject and difficulty
- **Detailed Results** - Comprehensive performance analytics
- **Progress Tracking** - Monitor your improvement over time
- **Question Review** - Review answers with detailed explanations
- **Mobile Friendly** - Access from any device

### System Requirements

- **Browser:** Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet:** Stable connection required
- **Device:** Computer, tablet, or smartphone

## Student Guide

### Creating an Account

1. Visit the UTME Master website
2. Click "Register" button
3. Fill in your details:
   - First Name
   - Last Name
   - Email Address
   - Password (minimum 8 characters)
4. Click "Register"
5. Check your email for confirmation
6. Log in with your credentials

### Logging In

1. Go to the login page
2. Enter your email and password
3. Click "Login"
4. You'll be redirected to your dashboard

### Dashboard Overview

Your dashboard shows:
- **Total Exams Taken** - Number of exams completed
- **Average Score** - Your average performance
- **Pass Rate** - Percentage of exams passed
- **Recent Exams** - Last 5 exams taken
- **Progress Overview** - Subject-wise performance

### Taking an Exam

1. Navigate to "Select & Start Exam"
2. Browse available exams
3. Click "Start" on your chosen exam
4. Read instructions carefully
5. Click "Start Exam"
6. Answer questions one by one
7. Use "Next" to move forward
8. Use "Previous" to review answers
9. Click "Submit Exam" when finished

### Practice Tests

1. Navigate to "Practice Tests"
2. Select subject and difficulty level
3. Choose number of questions
4. Set time limit
5. Click "Start Practice Test"
6. Complete the test
7. View results and analytics

### Viewing Results

1. Go to "Results" page
2. Select exam from history
3. View:
   - Overall Score
   - Subject Breakdown
   - Performance Analytics
   - Question Review

### Reviewing Answers

1. From results page, click "Review Answers"
2. View each question with:
   - Your answer
   - Correct answer
   - Explanation
   - Time spent

### Sharing Results

1. From results page, click "Share"
2. Choose platform:
   - Facebook
   - Twitter
   - LinkedIn
   - Copy Link
3. Share with friends or on social media

### Password Reset

1. On login page, click "Forgot Password"
2. Enter your email
3. Check your email for reset link
4. Click link and set new password

## Teacher Guide

### Creating an Account

Teachers register the same way as students but with role "TEACHER".

### Dashboard Features

- **Student Progress** - Monitor student performance
- **Question Bank** - Manage questions
- **Exam Creation** - Create custom exams
- **Analytics** - View class performance

### Managing Questions

1. Navigate to "Question Management"
2. Click "Create Question"
3. Fill in:
   - Question Text
   - Question Type (MCQ, True/False, etc.)
   - Options
   - Correct Answer
   - Difficulty Level
   - Subject
4. Click "Save"

### Creating Exams

1. Navigate to "Exam Management"
2. Click "Create Exam"
3. Fill in:
   - Exam Title
   - Description
   - Duration (minutes)
   - Total Marks
   - Pass Marks
   - Subject(s)
   - Number of Questions
4. Click "Save"
5. Publish exam when ready

### Viewing Student Progress

1. Navigate to "Analytics"
2. Select student from dropdown
3. View:
   - Exam history
   - Score trends
   - Subject performance
   - Improvement metrics

## Feature Documentation

### Exam Types

#### Official Exams
- JAMB, WAEC, NECO past questions
- Real exam format
- Timed sessions
- Official scoring

#### Practice Tests
- Customizable by subject
- Customizable by difficulty
- Customizable by number of questions
- Customizable time limit

### Question Types

1. **MCQ** (Multiple Choice)
   - Single correct answer
   - Four options (A, B, C, D)

2. **True/False**
   - Two options
   - True or False

3. **Fill in the Blank**
   - Text input
   - Case-insensitive matching

4. **Essay**
   - Long-form answers
   - Manual grading

### Scoring System

- **Correct Answer:** Full points
- **Incorrect Answer:** 0 points
- **Unanswered:** 0 points

### Grading Scale

| Percentage | Grade |
|------------|-------|
| 90-100% | A+ |
| 80-89% | A |
| 70-79% | B+ |
| 60-69% | B |
| 50-59% | C+ |
| 40-49% | C |
| 30-39% | D |
| 0-29% | F |

### Pass/Fail Criteria

- **Pass:** Score >= Pass Marks
- **Fail:** Score < Pass Marks

## FAQ

### General Questions

**Q: Is UTME Master free?**
A: UTME Master offers a free trial with limited features. Premium plans are available for full access.

**Q: Can I take exams on mobile?**
A: Yes, UTME Master is fully responsive and works on all devices.

**Q: Are results saved?**
A: Yes, all results are saved and can be viewed anytime.

**Q: Can I retake exams?**
A: Yes, if the exam allows retakes. Check exam settings.

**Q: How long are exams?**
A: Exam duration varies. Check exam details before starting.

### Technical Questions

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, and Edge (latest versions).

**Q: What if I lose internet during an exam?**
A: Your progress is saved automatically. Resume when connection is restored.

**Q: How do I contact support?**
A: Email support@utmemaster.com or use the help center.

### Account Questions

**Q: How do I change my password?**
A: Go to profile settings and click "Change Password".

**Q: How do I update my profile?**
A: Go to profile settings and edit your information.

**Q: Can I delete my account?**
A: Yes, go to settings and click "Delete Account".

### Exam Questions

**Q: Can I skip questions?**
A: Yes, use "Next" to skip and return later.

**Q: Can I change answers?**
A: Yes, use "Previous" to review and change answers.

**Q: What happens when time runs out?**
A: Exam is automatically submitted with answered questions.

**Q: Can I see correct answers after exam?**
A: Yes, after submission, view "Review Answers".

### Payment Questions

**Q: How do I upgrade to premium?**
A: Go to settings and select a premium plan.

**Q: What payment methods are accepted?**
A: Credit/Debit cards, bank transfer, and mobile money.

**Q: Is my payment secure?**
A: Yes, we use industry-standard encryption.
